

# 🚀 Prompt Library Project

✨ A modern and beginner-friendly AI Prompt Library built using React.js.

This project allows users to browse, search, and copy useful AI prompts easily.  
The project is simple, fast, responsive, and focused on clean UI design. 🎨

🌐 **Live Demo:**  
https://baighuzaif563-tech.github.io/Prompt-Library-Project/#/

---

# 📌 Features

✅ Browse AI Prompts  
✅ Search Prompts Quickly 🔍  
✅ Copy Prompt with One Click 📋  
✅ Category-wise Prompt Display 📂  
✅ Responsive UI 📱💻  
✅ Simple & Clean Interface ✨

# ⚡ Installation

Clone the repository:

```bash
git clone https://github.com/baighuzaif563-tech/Prompt-Library-Project.git
```

Open project folder:

```bash
cd Prompt-Library-Project
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```
